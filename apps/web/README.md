# Web App

`apps/web` will contain the CopilotKit operator interface.

Primary responsibilities:

- creative brief editor,
- batch launch action,
- live event feed,
- candidate grid,
- audio preview controls,
- score breakdown,
- approve, reject, request variant, and select final actions,
- Weave trace links.

The web app should make the human curator part of the agent loop, not a passive observer.
