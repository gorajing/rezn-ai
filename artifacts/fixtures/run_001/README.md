# Fixture Run 001

This fixture lets teammates and judges run the REZN Conductor loop without
Ableton.

The before pass is intentionally quieter, narrower, and muddier in the low mids.
The after pass is closer to the LUFS target, wider, and cleaner in the low-mid
band.

The WAV files are short generated tones. Replace them with real Ableton renders
when Jin captures live demo artifacts.
