# Sponsor Integrations

## W&B Weave

Primary target.

Weave is the proof layer:

- trace tree for every agent/tool/scorer step;
- custom scorers for audio quality and iteration delta;
- before/after metrics attached to the run;
- link included in the final submission.

## OpenAI

OpenAI powers structured specialist agents:

- Composer
- Critic
- Mix Engineer
- Conductor

Agents return JSON. The backend does not scrape prose.

## Redis

Redis stores:

- run state;
- event stream;
- remembered lessons;
- recall for the next run.

## CopilotKit

CopilotKit provides the control room:

- current run context;
- approval/reject actions;
- human-in-the-loop for taste-changing fixes.

## Cursor

Cursor is a build accelerator, not a runtime dependency.
